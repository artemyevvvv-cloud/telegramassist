### Модуль 6, Урок 5 — Трёхуровневая память агента: метод Карпатого

**Инструменты:**
- [Obsidian](https://obsidian.md) — просмотр и редактирование wiki
- [claude-memory-compiler](https://github.com/coleam00/claude-memory-compiler) — хуки + скрипты для памяти
- [Obsidian Web Clipper](https://obsidian.md/clipper) — сохранять статьи из браузера в `raw/`
- [Пост Карпатого — первоисточник метода](https://gist.github.com/karpathy/442a6bf555914893e9891c11519de94f)

**Промпты:**
- Установить систему памяти: *«Установи claude-memory-compiler в этот проект: https://github.com/coleam00/claude-memory-compiler Настрой три хука: SessionStart, SessionEnd, PreCompact. Создай структуру папок raw/ и wiki/. Создай CLAUDE.md с инструкцией читать wiki в начале работы.»*
- Настроить wiki под проект: *«У меня проект: [опиши одним предложением]. Что тебе нужно помнить между сессиями? Предложи структуру wiki и заполни первые версии.»*
- Подключить одну wiki к нескольким проектам: добавить в CLAUDE.md нового проекта путь к wiki.

**Скрипты:**
```
uv run python scripts/compile.py   # скомпилировать логи в wiki
uv run python scripts/query.py "вопрос"  # спросить базу знаний
uv run python scripts/lint.py      # проверить здоровье базы
```

### Модуль 7, Урок 2 — Подключаем домен

**Где купить:**
- [hoster.kz](https://hoster.kz) — любые карты, удобный
- [porkbun.com](https://porkbun.com) — дешёвые .com
- [reg.ru](https://reg.ru) — российские карты
- [namecheap.com](https://namecheap.com) — большой выбор зон

**При оформлении:**
- WHOIS Privacy Protection — включить
- Платный SSL — не нужен, Vercel выдаст бесплатно

**DNS-записи для Vercel:**
```
A     @    76.76.21.21
CNAME www  cname.vercel-dns.com
```
Точные значения: Vercel → Settings → Domains (они уникальные).  
DNS распространяется 5–60 мин, иногда до 24 ч. Проверить: [dnschecker.org](https://dnschecker.org)
