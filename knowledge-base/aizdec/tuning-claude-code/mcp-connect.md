## Что за чудо такое 🔌

MCP это типа USB-C для AI. Универсальный способ подключить Claude к внешним сервисам. Стандарт подключения.

![image](https://api.aizdec.app/storage/v1/object/public/lesson-media/uploads/1772121770747-image.png)

**😢** Без MCP: система видит только файлы на твоём компьютере.

**🥳** С MCP: видит твои базы данных, приложения, сайты и работает с ними напрямую.

---

## Что так можно соединить

**Мессенджеры и почту 💬**

[ x ] Telegram, Slack, Discord

[ x ] Gmail, WhatsApp

**Хранилища данных 📁**

[ x ] Google Sheets, Notion, Airtable

[ x ] Базы данных (PostgreSQL, MongoDB, Supabase)

**Бизнесовые приблуды 🔎**

[ x ] CRM-системы

[ x ] Трекеры задач

[ x ] Платёжные системы


**Свой сервис 🛸**

[ x ] Можно создать своё приложение и сделать для него MCP

[ x ] Или для телеграм-бота, чтобы агент мог им управлять

**В целом, это универсальная штука. Если какой-то сервис существует — скорее всего для него либо есть MCP, либо его несложно создать.**

## Когда нужен MCP

Когда данные где-то снаружи:

- Клиенты в Google Sheets или Notion

- Сообщения в Telegram боте

- Заказы в базе данных

- Задачи в трекере

Если хочешь чтобы Claude с ними работал напрямую и быстро, надо делать 🤘🏽

---

## Как начать

Бля, изи. Скажи Клоду:

_Мне нужен MCP для X. Подключи, чтобы ты видел мои таблицы. Мне это нужно чтобы ..._

**///**

_Сделай MCP подключение к Y. Нужно чтобы ты видел мои файлы и мог их редактировать._

Далее потребуется внести некоторые уточнения, дать ему больше деталей, но в целом для Клода это совершенно не сложная задача, и он с ней легко справляется. Поэтому не вижу смысла растягивать инструкцию на длинющее полотно.

---

## Что меняется после подключения

**[ - ] Без MCP:** надо копировать данные → вставлять в чат → повторять что с ними надо делать → постоянно прыгать из вкладки во вкладку.

**[ + ] С MCP:** пишешь _"посмотри таблицу клиентов и найди должников"_. Claude сам открывает, сам ищет, и если надо – сразу отправляет им сообщения.

**Постарался объяснить максимально доступным языком :)**

## Важно про безопасность

Если не ограничить MCP в действиях, Claude Code получит полный доступ к подключенному сервису и твоим данным. Если это чужой MCP, то он ещё и может отправить эти данные куда-нибудь.

Поэтому будь внимателен.

## MCP + Commands

Помнишь команды из прошлого блока? Команды становятся мощнее когда у Claude Code есть доступ к данным через MCP.

Можно сделать так, чтобы по определенной команде, агент загружал список невыполненных задач из трекера, проверял новые входящие письма, данные с твоих умных часов > и выдавал тебе через минуту расширенный отчёт о состоянии твоего здоровья и бизнеса.

**В команде это выглядит так:**

```text
Загрузи задачи через get_tasks()
Загрузи входящие через get_inbox()
Покажи отчёт в таком формате:
...
```

Клод увидит названия инструментов и поочереди будет вызывать их через MCP.
