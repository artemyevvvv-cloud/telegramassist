## Три модели 🧠

В Claude Code доступны три типа мозгов. Переключаться между ними можно командой `/models` [появляется список, выбираешь нужную]. Познакомимся с каждой:

![image](https://api.aizdec.app/storage/v1/object/public/lesson-media/uploads/1772121148308-image.png)

**Opus 4.6** — МЕГА-БОСС 😈

Вышла 5 февраля 2026. Лучше всех справляется со сложными задачами: длинные цепочки рассуждений, работа с большими проектами, нестандартные ситуации. Стоит дороже, работает медленнее, но результат заебательский. Пока лучше всего, что есть на рынке.

_У Opus есть __**Fast Mode**__ — тот же мозг, но отвечает быстрее. Работает только в терминале (не в VS Code). Включается командой _`/model`_ → выбираешь Opus 4.6 fast._

**Sonnet 4.6** — СЕРЕДНЯЧОК 🥷🏽

Быстрая, достаточно умная для большинства задач. Наман для рутинной несложной работы: если над написать текст, быстро разобраться в  коде, обработать не_научные данные. Для важных задач не советую.

**Haiku 4.5** — МАЛЫШ 👼🏽

Может переформатировать текст, быстро и коротко ответить на вопрос, сделать что-то по шаблону. Думает не очень, но если действия заранее чётко прописаны, то вывезет. Стоит копейки ваще.

## Контекстное окно 🗑️

Контекстное окно это объем инфы, которую Claude может держать в своей цифровой башке одновременно. У «Opus 4.6» и «Sonnet 4.6» оно выросло до 1 миллиона токенов (раньше было 200 тысяч).

Что эт значит: если раньше Claude мог видеть условно ~300 страниц текста за раз, теперь ~1500. Братишка может одновременно читать гораздо больше файлов и не закипать, и не терять нить беседы.

## Подписка или API-ключ 💸

**[ 1 ] Подписка** нужна для личного использования. Открыл VS Code, работаешь с Клодом, закрыл. Всё.

**[ 2 ] API-ключ** для автоматизаций. Когда нужно чтобы агенты работали сами, без тебя. Например: каждое утро собирать отчёт, каждый час проверять что-то, запускать задачи по расписанию.

Действия Claude Code, которого ты используешь по подписке, тоже можно автоматизировать – но не во всех случаях. В следующих уроках мы поговорим об этом подробнее.

## Remote Control 📱

С февраля 2026 появилась фича «Remote Control» — можно управлять Claude Code с телефона. Запускаешь `claude remote-control` в терминале, сканируешь QR-код, и открывается сессия на claude.ai/code прямо в мобильном браузере. Работает через защищённое соединение, требует Max подписку.

**Но важно понимать: **это фича для разработчиков. Ты буквально видишь терминал на телефоне. Если хочешь управлять рабочими процессами: разработкой, продажами, маркетингом, с телефона через удобный интерфейс =  нужен другой, более серьёзный подход. Об этом поговорим в конце курса.
