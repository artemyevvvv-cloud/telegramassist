# Шаг 1: попробуй стандартную установку

Открой на ноутбуке приложение «Терминал» (на Mac найди через Spotlight: Cmd+Пробел → набери "Terminal").

![image](https://ajvmyjuqfdjrljmodbdb.supabase.co/storage/v1/object/public/lesson-media/uploads/1773196608955-Screenshot_2026-03-11_at_05.36.44.png)

### **Вставь команду:**

```text
curl -fsSL https://claude.ai/install.sh | bash
```

**Если пошла установка > отлично, переходи к шагу 4.**

**Если в ответ пришло что-то вроде:**

```text
<html>
<head><title>302 Found</title></head>
<body>
<center><h1>302 Found</h1></center>
<hr><center>cloudflare</center>
</body>
</html>
```

Или длинное сообщение с текстом «App unavailable in region» = значит Claude заблокирован ☠️ в твоей стране. **Тогда переходи к шагу 2****. **Если в терминале просто тишина и ничего не произошло = тоже заблокирован.

# Шаг 2: установи Homebrew 📦

Homebrew это как магазин приложений, только через терминал. Им пользуются миллионы разработчиков, он безопасный и бесплатный. Скачивается с GitHub, а GitHub в России не заблокирован.

**Вставь в терминал:**

```text
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Терминал попросит 🔒 пароль от Mac (тот, которым ты разблокируешь ноутбук). При вводе пароля символы не отображаются, это нормально, просто набери и нажми Enter.

Потом спросит подтверждение > нажми Enter. Не ссы!

Дальше начнётся установка. Она может занять 5-15 минут — будет скачивать Xcode Command Line Tools. Просто жди. **Когда всё закончится, увидишь:**

```text
==> Installation successful!
```

И** вернётся обычная строка ввода** (`username@MacBook ~ %`). **Значит готово.**

---

**Важно:** после установки терминал ещё не знает где Homebrew. Вставь эту команду, она «включит» его в текущей сессии:

```text
eval "$(/opt/homebrew/bin/brew shellenv zsh)"
```

Без этого при попытке ввести `brew` будет ошибка `command not found`. В следующий раз при открытии терминала эта команда уже не понадобится — Homebrew пропишется автоматически.

# Шаг 3: установи Claude Code через Homebrew

**Теперь вставь:**

```text
brew install --cask claude-code
```

Это скачает и установит Claude Code на твой Mac. Тоже качает с GitHub.

# Шаг 4: получи токен

🇷🇺 Включи VPN (Claude должен думать, что ты не в России, если ты там).

**Вставь в терминал:**

```text
claude setup-token
```

**Откроется 🌐 браузер. **Ты должен быть залогинен в аккаунт Claude, где у тебя оплачена подписка (Max за $100 или $200 в месяц).

**Нажми «Authorize».**

После этого тебя вернёт обратно в терминал. Там появится 🔑 жёлтая строка с токеном — это и есть твой ключ. **Скопируй его и отправь своему боту в Telegram.**
