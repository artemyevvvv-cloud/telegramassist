MCP-серверы = внешние процессы на VPS, которые добавляют агенту инструменты.

---

# MCP в меню

Главное меню → 🔌 MCP → 📂 workspace.

Здесь список подключённых серверов с типом (http/stdio). «➕ Добавить» — выбор пресета и ввод ключа. Карточка сервера показывает доступные инструменты (`mcp__servername__*`) и кнопку «🗑 Удалить».

MCP-серверы работают как процессы на VPS и потребляют оперативную память. Если сервер точно не будет использоваться, отключи его.

---

# Пресет Supadata

Транскрипция видео, метадата, веб-скрейпинг. Поддерживает YouTube, TikTok, Instagram, X, Facebook. 100 бесплатных запросов в месяц, от $5/мес.

**Подключение: **🔌 MCP → ➕ → Supadata → ввести ключ (начинается с `sd_`).
**Получить ключ: **dash.supadata.ai → регистрация → ключ в дашборде.

После подключения у агента появляются инструменты:

- `mcp__supadata__transcript` — транскрипция видео по URL

- `mcp__supadata__metadata` — метадата видео (заголовок, описание, длительность)

- `mcp__supadata__scrape` — чистый markdown с любой веб-страницы

- `mcp__supadata__extract` — извлечение структурированных данных

# Пресет Apify

23 тысячи скрейперов (акторов). Проверенные акторы для YouTube, Instagram, TikTok, Twitter. $5 бесплатно в месяц, от $29/мес.

**Подключение:** 🔌 MCP → ➕ → Apify → ввести токен (начинается с `apify_api_`). **Получить ключ:** console.apify.com → Settings → Integrations.

Подключённые серверы не отображаются в списке добавления.

Каждый сервис в меню показывает: где получить ключ, какой формат, какие права выбрать. Ключ сохраняется и работает сразу, без перезапуска.

**Для GitHub **рекомендуется fine-grained токен на конкретный репозиторий, не classic на весь аккаунт. Для Railway > токен проекта, не аккаунта.

Системные ключи (BOT_TOKEN, ANTHROPIC_API_KEY и ещё 9) защищены от перезаписи. Полный гайд по каждому сервису = файл `SERVICES.md` на VPS.
