# Шаг 2 — Привязываем оплату

![image](https://img2.teletype.in/files/da/b4/dab43d1f-455f-4ae6-beea-1f11860a20bd.jpeg)

⚠️ **ВАЖНО!
**Если на странице оплаты нет варианта «Криптовалюта» = твоя почта ассоциируется с Россией :/ DigitalOcean скрывает этот способ оплаты. 

### ЧТОБЫ ИСПРАВИТЬ:

**[1]** **Активируй номер на Give SMS** >>> выбираешь там страну, в сервисах находишь Gmail (даётся на 15 минут, их хватит на все шаги)

**[2]** **Создай новую почту через мобильное приложение Gmail,** не через браузер. При регистрации страна VPN должна совпадать с арендованным номером (по завершении регистрации можешь спокойно войти в неё через ПК)

**[3]** **Скачай приложение Antigravity, войди под новой почтой** >>> оно попросит верифицировать аккаунт через СМС. Введи тот же номер, подтверди код. Без этого почту позже могут заблокировать

**[4]** Готово =)  Регистрируешься на DigitalOcean с новой почтой, и на странице оплаты появляется возможность привязать криптокошелёк! 

![image](https://img1.teletype.in/files/87/0f/870f683b-72a1-4276-a5b7-04fda207188e.png)

Если выбрал карту — адрес в форме должен совпадать с биллинг-адресом карты. Если находишься не в стране, где карта выпущена, включи VPN в эту страну.

![image](https://img3.teletype.in/files/e4/37/e4376fe8-9ae4-4b21-b3c7-e2377a19cc08.png)

Если платишь криптой, можно выбрать любой адрес.

Если выбрал **крипту** — в строке адреса начинаем вводить пару [любых] символов, сайт подскажет и заполнит остальные поля автоматически.

![image](https://img1.teletype.in/files/0a/c3/0ac3fbb1-195b-41da-bc5a-ff694eed727c.png)

Следующий шаг для крипты:** привязать кошелёк. **Выбираем свой, в моём случае MetaMask. Для оплаты нужен **USD Coin в сети Base** или Polygon. Если его нет, в самом кошельке можно обменять любую крипту на USD Coin в пару кликов.

![image](https://img4.teletype.in/files/33/df/33df205f-a2d0-4288-93c6-cd0a80dc2c3c.png)

Жмём **Save and continue**

# Шаг 3 — Получаем API токен

В левом меню листаешь в самый низ → **API** → **Generate New Token**

![image](https://img1.teletype.in/files/ce/f7/cef76b28-7038-4f2f-8021-c6b84512fa18.jpeg)

![image](https://img1.teletype.in/files/8d/a8/8da8089f-aeae-4562-bf43-4b3617598309.jpeg)

**Token Name** — любое

**Expiration** — No expiry

**Scopes** — Full Access

![image](https://img4.teletype.in/files/b5/51/b55159be-5954-4ed6-af35-22096240d65e.jpeg)

Жмёшь **Generate Token**
