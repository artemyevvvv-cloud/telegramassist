Этот воркфлоу представляет собой простую систему для автоматического создания трендовых ASMR-видео. Процесс организован так, чтобы вы могли легко генерировать контент, следуя всего одному простому шагу: **внести данные в Google Таблицу.**

Вам достаточно указать базовые идеи (например, какое вещество, какая жидкость и какая реакция должны произойти), а система сделает все остальное. Встроенный AI-ассистент на базе OpenAI (GPT-4o) самостоятельно превратит ваши короткие идеи в детализированный и красочный сценарий для видео. После этого новая, быстрая и экономичная модель **VEO 3 Fast** от сервиса [**Fal.ai**](http://fal.ai/) сгенерирует из этого сценария готовый видеоролик. В конце процесса ссылка на видео автоматически появится в вашей таблице.

Ключевое отличие этого воркфлоу от self-hosted версий — его **простота и доступность**. Вам не нужно настраивать собственный сервер, устанавливать Python-библиотеки или работать с командной строкой. Вся сложная работа по генерации видео выполняется надежными облачными сервисами.

Таким образом, этот шаблон — идеальная отправная точка для тех, кто хочет быстро и без лишних технических сложностей запустить массовое производство качественных AI-видео.

### **Объяснение каждого столбца:**

- `**substance**`** (вещество):**
Основной объект, на который будет капать жидкость.

_Пример:_ `*fine gray powder` (мелкий серый порошок)*

- `**liquid**`** (жидкость):**
Жидкость, которую капают из пипетки.

_Пример:_ `*glowing green liquid` (светящаяся зеленая жидкость)*

- `**reaction**`** (реакция):**
Что происходит в момент контакта.

_Пример: _`_magical reaction_`_ (магическая реакция)_

- `**transformation**`** (трансформация):**
Во что превращается вещество в итоге.

_Пример: _`_vibrant voxel-style grass and trees_`_ (яркая трава и деревья)_

- `**art_style**`** (стиль):**
Общий художественный стиль видео.

_Пример:_ `*Minecraft*`

Эти пять столбцов — твоя палитра для AI. Воркфлоу берет эти слова и собирает из них красивый, сложный сценарий для видео. **Вот ещё два важных пункта:**

- `**Status**`** (статус):**
Это **ключевой столбец** для управления процессом.

**Как работает:**

Укажи здесь `Pending` для тех строк, которые нужно обработать. Воркфлоу будет искать именно это слово, чтобы взять задачу в работу. После успешной генерации видео, он автоматически изменит статус на `Complete`.

**Совет:** сделай этот столбец с выпадающим списком ("Pending", "Complete") для удобства.

- `**Output**`** (результат):** Этот столбец **оставь пустым**. Воркфлоу сам заполнит его, вставив прямую ссылку на готовое видео после генерации.
