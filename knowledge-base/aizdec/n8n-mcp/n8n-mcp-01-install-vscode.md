# Программа VS Code

**Курс:** N8N + Claude Code  
**URL:** https://www.aizdec.me/courses/n8n-mcp/n8n-mcp-01-install-vscode

---

VS Code — редактор кода от Microsoft. Он бесплатный. Claude Code работает внутри него как расширение. **Взаимодействие происходит через чатик. **Терминал открывать не нужно, программировать и писать сложные команды – тоже. Устанавливается за 5 мин. Погнали!

## **[ СКАЧИВАНИЕ ]**

1. Открой **code.visualstudio.com**

1. Нажми большую синюю кнопку **Download**

1. Сайт сам определит твою систему — Mac или Windows

_Файл весит около 100 МБ, скачается за пару минут._

## **[ УСТАНОВКА НА MAC ]**

1. Скачается файл `.zip`

1. Двойной клик — распакуется

1. Перетащи иконку VS Code в папку **Программы** (Applications)

1. Запусти из папки **Программы**

⚠️ **Если Mac не даёт открыть:
**Правый клик на иконке → Открыть → Подтвердить. Mac иногда блокирует программы, скачанные из интернета. Это нормально.

## **[ УСТАНОВКА НА WINDOWS ]**

1. Скачается файл `.exe`

1. Запусти его

1. Следуй шагам установщика

1. На экране "Select Additional Tasks" поставь галочку **"Add to PATH"**

Галочка _"Add to PATH" _важна — без неё Claude Code не сможет работать с терминалом.

## **[ ПЕРВЫЙ ЗАПУСК ]**

Запусти VS Code. Увидишь тёмное окно с приветствием и панелью иконок слева.

Поздравляю! Редактор для кода установлен. Теперь займёмся Claude Code.

💭 **От Тома:
**Неподготовленного челика интерфейс VS Code может испугать, но за пару дней это ощущение пройдёт. Моя прабабка говорила: "Глаза боятся – руки делают". Вот эта фраза здесь подходит лучше всего.
