# Что такое MCP

**Курс:** N8N + Claude Code  
**URL:** https://www.aizdec.me/courses/n8n-mcp/n8n-mcp-10-what-is-mcp

---

В прошлом модуле ты установил Claude Code. Он умеет работать с кодом, видит файлы проекта, может их редактировать.

Но про N8N он знает в общих чертах. Спроси_ "какие параметры у HTTP Request ноды в Н8Н?"_ — получишь общий ответ. Возможно неправильный. Вероятнее всего, устаревший.

MCP это чинит.

## **[ ЧТО ЗА ЧУДО ТАКОЕ ]**

Это способ дать Claude дополнительные знания и инструменты. Подключаешь MCP-сервер — он получает доступ к его возможностям. Супер-сила, проще говоря.

**Мы используем N8N-MCP — он хранит в себе:**

- Документацию по каждой ноде

- Готовые шаблоны (тысячи)

- Инструменты работы с ошибками

- Возможность создавать шаблоны

- Возможность управлять конструктором

## **[ ЧТО ПОЛУЧАЕШЬ ]**

Друга, наставника, товарища, сотрудника, помощника – и всё в одном лице за $20-200 в месяц.

## **[ КАК РАБОТАЕТ ]**

1. Открываешь VS Code с папкой проекта

1. N8N-MCP запускается автоматически в фоне

1. Claude получает доступ к инструментам N8N

1. Можешь сразу задавать вопросы и давать задачи

**Один раз настроил — работает при каждом запуске. Изи!**

## **[ API КЛЮЧ ОТ N8N ]**

Базовые функции этого MCP работают без подключения к твоему N8N. Но если воткнуть ключ, агент сможет видеть загруженные шаблоны, историю запусков, ошибки – и полноценно управлять тем, что у тебя там происходит 🔧

А если не втыкать ключ, то твой агент получит актуальную базу знаний по N8N, с которой ему будет комфортно работать, и в которой он не потеряется. Это позволит тебе заиметь умного советника, (реально) знающего ответ на любой вопрос 🧠
