# Расширение Claude Code

**Курс:** N8N + Claude Code  
**URL:** https://www.aizdec.me/courses/n8n-mcp/n8n-mcp-02-install-claude-code

---

Claude Code — это агент для работы с кодом. Его можно использовать либо через терминал (сложно и неудобно, для задротов) / либо через расширение для VS Code. Есть и другие способы, но эти два – самые выгодные с точки зрения затрат.

## **[ УСТАНОВКА РАСШИРЕНИЯ ]**

1. Открой VS Code

1. Нажми на иконку **Extensions** слева (четыре квадратика)
- Или: `Cmd+Shift+X` (Mac) / `Ctrl+Shift+X` (Windows)

1. В поиске напиши: **Claude Code**

1. Найди расширение от **Anthropic** (официальное)

1. Нажми **Install**

Установка займёт несколько секунд.

## **[ АВТОРИЗАЦИЯ ]**

После установки нужно войти в аккаунт. Он либо предложил сам, либо сделай так:

1. Нажми `Cmd+Shift+P` (Mac) или `Ctrl+Shift+P` (Windows)

1. Напиши: **Claude Code: Open**

1. Выбери первый результат

1. Появится панель Claude Code справа

1. Нажми **Sign In**

1. Откроется браузер — войди в аккаунт Anthropic

1. Подтверди доступ

Вернёшься в VS Code — панель Claude Code будет готова к работе.
