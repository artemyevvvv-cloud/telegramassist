**Курс:** Публикация проекта в инет  
**URL:** https://www.aizdec.me/courses/publish/setup-nowpayments

Сейчас настроим первую платежную систему криптовалютных платежей.

## [ ПОЧЕМУ NOWPAYMENTS ]

Она быстрее всего настраивается, поэтому идеально подходит для примера. А, ну и принимает огромное количество монет в разных сетях. 

**Когда использовать:**

- Нужны международные платежи без ограничений по странам

- Ты не можешь работать с обычными платежными системами (ну мало ли)

- Твой бизнес в "серой зоне" среди норм платёжек

**[+] Плюсы:**

- Нет чарджбеков (транзакция необратима)

- Работает в любой стране

- Быстрая регистрация (не требует документов для старта)

- Низкие комиссии (0.5-1%)

**[-] Минусы:**

- Клиенту нужен криптокошелек (не все умеют)

- Курс меняется (решение: используй USDT — стейблкоин привязанный к доллару)

## [ ШАГ 1: РЕГИСТРАЦИЯ ]

1. Открой сайт nowpayments.io

1. Нажми "Sign Up" (Зарегистрироваться)

1. Введи email и пароль

1. Подтверди email (придет письмо со ссылкой)

**Всё!** Регистрация готова. Не помню насчет необходимости верификации по паспорту, но кажется что её там нет вообще, либо есть на очень больших оборотах.

## [ ШАГ 2: ПОЛУЧАЕМ API КЛЮЧ ]

После входа в личный кабинет:

1. Открой раздел **Settings** (Настройки)

1. Найди **API Keys**

1. Нажми **Generate API Key** (Создать API ключ)

1. Скопируй ключ (выглядит как: `ABC123-DEF456-GHI789...`)

**Важно:** API ключ нужен для создания платежей через код.

**Куда сохранить:**

Добавь в файл `.env`:

```text
NOWPAYMENTS_API_KEY=твой-ключ-здесь
```

И в Vercel Environment Variables (Settings → Environment Variables → Add):

```text
Name: NOWPAYMENTS_API_KEY
Value: твой-ключ-здесь
```

---

## [ ШАГ 3: IPN SECRET ]

IPN Secret — это секретное слово, которое знаете только ты и NowPayments. Оно нужно для проверки, что уведомление о платеже пришло именно от NowPayments (а не от мошенника).

**Где взять:**

1. В настройках NowPayments открой раздел **IPN Settings**

1. Найди **IPN Secret Key**

1. Если его нет — нажми **Generate**

1. Скопируй ключ

**Куда сохранить:**

В `.env`:

```text
NOWPAYMENTS_IPN_SECRET=твой-секретный-ключ
```

В Vercel Environment Variables:

```text
Name: NOWPAYMENTS_IPN_SECRET
Value: твой-секретный-ключ
```

**⚠️ ВАЖНО:
**Никому не показывай IPN Secret! Если кто-то узнает этот ключ, он сможет подделать уведомления о платежах.

---

## [ ШАГ 4: SANDBOX — ТЕСТ РЕЖИМ ]

Sandbox (песочница) это **тестовый режим** для проверки платежей. Работает так, будто ты играешь в магазин с игрушечными бумажками. Все как в реальности, только деньги понарошку.

**Зачем нужен:**

- Проверить что код работает

- Протестировать webhook (уведомление о платеже)

- Убедиться что всё настроено правильно

- Только ПОСЛЕ тестов включить настоящие платежи

**Как включить Sandbox:**

1. В настройках NowPayments найди **Sandbox Mode**

1. Переключи на **Enabled** (Включено)

1. Получишь отдельные тестовые ключи:

- Sandbox API Key

- Sandbox IPN Secret

**Важно:** у sandbox свои ключи (отдельные от полноценных).

Добавь их в `.env`:

```text
NOWPAYMENTS_SANDBOX_MODE=true
NOWPAYMENTS_SANDBOX_API_KEY=тестовый-ключ
NOWPAYMENTS_SANDBOX_IPN_SECRET=тестовый-секрет
```

## [ ШАГ 5: УКАЗЫВАЕМ АДРЕС КРИПТОКОШЕЛЬКА ]

**Куда будут приходить деньги:**

NowPayments получает платеж от клиента и переводит сразу тебе. Нужно указать адрес твоего криптокошелька.

1. В настройках найди **Payout Settings** (настройки выплат)

1. Выбери криптовалюту (например USDT — стейблкоин)

1. Укажи адрес своего кошелька.

**Если у тебя нет криптокошелька:**

Самый простой способ это зарегаться на бирже:

- Binance (binance.com)

- Bybit (bybit.com)

- Или, если не любишь верифы по паспорту, используй SafePal.

Там получишь адрес для получения USDT.

**⚠️ В Sandbox режиме:** можешь указать любой адрес (деньги всё равно не настоящие).

---

## [ ШАГ 6: СОЗДАЕМ ТЕСТОВЫЙ ПЛАТЕЖ ]

Проверим что всё работает:

1. Открой sandbox.nowpayments.io (тестовый режим)

1. Войди с теми же данными

1. Нажми **Create Payment** (Создать платеж)

1. Заполни:

- Amount: 10 USDT

- Order ID: test_001

1. Нажми **Create**

Получишь страницу оплаты с QR-кодом. В sandbox режиме платеж подтвердится автоматически через 1-2 минуты (не нужно реально платить).

## [ ШАГ 7: НАСТРАИВАЕМ WEBHOOK URL ]

Webhook URL это адрес на твоем сайте, куда NowPayments отправит уведомление _"Клиент заплатил!"._

**Где указать:**

1. Открой Settings → **IPN Settings**

1. Найди поле **IPN Callback URL**

1. Укажи адрес: `https://твой-сайт.vercel.app/api/webhook`

**Для Sandbox:**

В sandbox режиме укажи тот же адрес (или отдельный для тестов).

**Пример:**

```text
Production: https://мой-сайт.vercel.app/api/webhook
Sandbox: https://мой-сайт.vercel.app/api/webhook
```
