# Stripe: международные платежи

**Курс:** Публикация проекта в инет  
**URL:** https://www.aizdec.me/courses/publish/stripe-international

---

Stripe это мировой стандарт для приёма платежей. Если есть возможность подключить его, подключай! Только вот возможность такая есть далеко не у всех..

## **[ КОГДА ИСПОЛЬЗОВАТЬ ]**

**[+] Подходит если:**

- Компания зарегистрирована в США, Европе, UK или Сингапуре

- Банковский счёт в этой стране

- Международная аудитория

**[-] Не подходит если:**

- Живёшь в РФ, Беларуси или Украине

- Нет зарубежной компании

- У тебя только русскоговорящая аудитория

Stripe не регистрирует аккаунты на российский паспорт. Даже через VPN. Даже если укажешь европейский адрес. При верификации попросят документы, и развернут.

## **[ ЧТО УМЕЕТ СТРАЙПЫЧ ]**

1. **Оплата в один клик** — Apple Pay, Google Pay встроены

1. **Форма на сайте** — клиент не уходит на другой домен

1. **Подписки** — автоматические списания, триалы, промокоды

1. **Автоматические налоги** — считает VAT для разных стран

## **[ КАК ПОДКЛЮЧИТЬ ]**

**Шаг 1. Регистрация**

Заходишь на stripe.com, создаёшь аккаунт. Понадобится:

- Данные компании (название, адрес, налоговый номер)

- Банковский счёт для вывода

- Документы директора (паспорт, подтверждение адреса)

**Шаг 2. Получаешь ключи**

В Dashboard → Developers → API keys:

- Publishable key — для фронтенда (можно показывать)

- Secret key — для сервера (никому не показывай)

- STRIPE_WEBHOOK_SECRET — в Webhooks

**Шаг 3. Интеграция**

**Попроси Claude Code:**

_Настрой Stripe Checkout для моего сайта.
Создай api/create-checkout-session.ts
Кнопка "Купить" которая редиректит на оплату
Страницы success и cancel._

Он создаст всё сам. Тебе останется добавить ключи в Vercel.

**Шаг 4. Webhook**

**Чтобы узнавать об оплатах, скажи Клоду:**

_Добавь webhook для Stripe.
Когда приходит событие checkout.session.completed:
- Проверь подпись
- Обнови статус пользователя в Supabase_

## **[ КОМПАНИЯ В США ОТ СТРАЙПА ]**

Если хочешь Stripe, но нет зарубежной компании, есть варики.

**Stripe Atlas**

_За $500 Stripe сам открывает тебе LLC в Делавере:_

- Регистрация компании

- Налоговый номер

- Банковский счёт

_Но потом платишь $100-300 в год за обслуживание плюс бухгалтера._

**[+] Имеет смысл если:**

- Оборот больше $5000 в месяц

- Международные клиенты

- Планируешь масштабироваться

**[-] Не имеет смысла если:**

- Только начинаешь

- Аудитория из РФ

- Оборот меньше $1000 в месяц
