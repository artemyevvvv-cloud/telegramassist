# Что такое хостинг и зачем нужен Vercel

**Курс:** Публикация проекта в инет  
**URL:** https://www.aizdec.me/courses/publish/what-is-hosting

---

Твой проект работает. На твоём компьютере. Открываешь localhost — всё отлично. Но это видишь только ты. Закрыл ноутбук, и сайт недоступен. Хочешь показать клиентам — никак.

Решение = хостинг.

## **[ ЧТО ТАКОЕ ХОСТИНГ ]**

Хостинг это место в интернете, где живёт твой сайт.

**Сейчас:**

- Твой проект на твоём компе

- Работает только когда он включён

- Видишь только ты

**С хостингом:**

- Проект живёт в интернете

- Работает 24/7 

- Доступен всему миру по ссылке

## **[ ПОЧЕМУ VERCEL ]**

![image](https://ajvmyjuqfdjrljmodbdb.supabase.co/storage/v1/object/public/lesson-media/uploads/1766454046904-Screenshot_2025-12-23_at_04.40.33.png)

**Бесплатный план**

Hobby-план бесплатный и щедрый:

- _100 ГБ трафика в месяц_

- _Неограниченное количество проектов_

- _HTTPS (защищённое соединение) бесплатно_

Для личных проектов и даже небольшого бизнеса хватит с запасом.

**Работает с GitHub**

Твой агент загрузил код в GitHub → Vercel сам всё собрал. Агент изменил код → сайт обновился автоматически. Никаких ручных действий.

**Быстрый**

Vercel раздаёт сайт с компьютеров по всему миру. Твой сайт будет загружаться быстро откуда угодно, хоть из Москвы, хоть из Нью-Йорка.

**Claude его знает**

Если что-то пойдёт не так — Claude поможет разобраться.

💡 **Альтернативы:
**Есть и другие похожие сервисы: Netlify, Cloudflare Pages. Принцип тот же: подключаешь GitHub, получаешь сайт. Мы используем Vercel, но если он не подойдёт, альтернативы работают +- так же.
