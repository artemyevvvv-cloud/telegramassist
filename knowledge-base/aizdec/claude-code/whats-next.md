### **Путь 2: прокачать Claude Code до максимума**

Ты знаешь основы. Но то, что ты видел это процентов 20 от возможностей Claude Code.

**Что ты ещё не трогал:**

- **Skills** — навыки, которые агент выполняет по команде

- **MCP** — подключение внешних сервисов (Supabase, GitHub, Telegram, что угодно)

- **Hooks** — автоматические действия без твоего участия

- **Worktrees** — параллельная работа над несколькими задачами одновременно

- **Agent Teams **— команды агентов, которые работают вместе

**Для кого:**

- Хочешь превратить Claude Code из помощника в систему

- Готов автоматизировать рутину

- Интересно, как другие люди строят рабочие процессы на Claude Code (разбираем реальные кейсы)

**– > Курс: ****Тюнинг Claude Code**
