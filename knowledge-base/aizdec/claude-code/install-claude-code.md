# Расширение Claude Code

**Курс:** Claude Code  
**URL:** https://www.aizdec.me/courses/claude-code/install-claude-code

---

VS Code стоит. Теперь добавим в него расширение Claude Code, которое превращает обычный редактор в AI-помощника.

## **[ ШАГ 1: ОТКРЫТЬ МАГАЗИН РАСШИРЕНИЙ ]**

1. Открой VS Code

1. Слева найди иконку с четырьмя квадратиками — **Extensions** (Расширения)

1. Кликни на неё

![image](https://ajvmyjuqfdjrljmodbdb.supabase.co/storage/v1/object/public/lesson-media/uploads/1766445375889-Screenshot_2025-12-23_at_02.16.11.png)

Откроется панель поиска расширений.

💡 **Горячая клавиша:
**Mac: `Cmd + Shift + X
`Windows: `Ctrl + Shift + X`

## **[ ШАГ 2: НАЙТИ И УСТАНОВИТЬ ]**

1. В поле поиска напиши **Claude Code**

1. Найди расширение от **Anthropic** (официальное)

1. Нажми синюю кнопку **Install.**

Установка займёт несколько секунд. После этого появится иконка Claude — оранжевый логотип. Она может быть слева в боковой панели или справа вверху.

⚠️ **Важно:
**Убедись что устанавливаешь именно от **Anthropic**. Есть подделки с похожими названиями. Смотри на издателя, должно быть написано "Anthropic".

## **[ ШАГ 3: АВТОРИЗАЦИЯ ]**

После установки нужно войти в свой аккаунт Anthropic. Заранее зарегистрируй его и оплати подписку.

> **Предупреждение: 
**Потребуется очень хороший VPN и покупка зарубежной сим-карты. Напиши ассистенту, он подскажет где их можно добыть. Большинство стандарных VPNов не работают с профессиональными инструментами типа Claude Code.

### **Как открыть панель Claude Code**

**Способ 1 — через Command Palette (рекомендую):**

![image](https://ajvmyjuqfdjrljmodbdb.supabase.co/storage/v1/object/public/lesson-media/uploads/1766445552607-Screenshot_2025-12-23_at_02.19.05.png)

1. Нажми **Cmd + Shift + P** (Mac) или **Ctrl + Shift + P** (Windows)

1. Напиши: **Claude Code: Open in Side Bat**

1. Нажми Enter — панель откроется справа

**Способ 2 — через иконку:**

1. Найди иконку Claude (оранжевый логотип), она может быть в боковой панели слева или справа

1. Напиши /login.

**Авторизация:**

1. В открывшейся панели нажми кнопку **Sign In**

1. Откроется браузер — войди в свой аккаунт Anthropic (или создай новый)

1. Подтверди доступ для VS Code

**После этого вернись в VS Code — ты авторизован.**

💡 **Совет:
**Если иконка Claude пропала или ты не можешь её найти, используй Command Palette (Cmd/Ctrl + Shift + P → _"Claude Code: Open in sidebar"_). 
**Либо есть комбинация клавиш, на маке это:
****Shift + Command + Escape**

## **[ ШАГ 4: ПРОВЕРКА ]**

**Убедимся что всё работает:**

1. Открой панель Claude Code (иконка с оранжевым логотипом)

1. Справа откроется чат, внизу поле для ввода текста

1. Напиши: _**Привет**_

1. Нажми Enter

Если Claude ответил — поздравляю, установка завершена!

**Проверка:**

✅ Иконка Claude Code появилась (справа вверху)

✅ Панель чата открывается справа

✅ Ты авторизован 

✅ Claude отвечает на сообщения

**Всё так? → Переходи к следующему уроку.**

## **[ ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ ]**

**Проблема: Claude не отвечает или показывает ошибку**

1. **Проверь интернет** — открой google.com. Если не грузится — проблема с сетью

1. **Проверь подписку** — зайди на claude.ai, посмотри активна ли подписка Pro/Max

1. **Перезапусти VS Code** — закрой полностью и открой заново

1. **Переустанови расширение** — удали Claude Code и установи заново

**Проблема: Кнопка Sign In не появляется**

1. Закрой VS Code полностью

1. Открой заново

1. Проверь свой VPN и качество сети

**Проблема: браузер открылся, но авторизация не работает**

1. Убедись что входишь в тот же аккаунт Anthropic где есть подписка

1. Если есть несколько аккаунтов — выйди из всех и войди в нужный

1. Попробуй другой браузер

💭 **От Тома:
**Если что-то пошло не так — не паникуй. РЕШАЕТСЯ ВСЁ. Если не получается, открой ChatGPT или Gemini, опиши проблему, скинь скриншотики. Когда голова закипит, перекури, или чайку попей. И опять за дело. Если дело в блокировках, уточни у ассистента что делать.
