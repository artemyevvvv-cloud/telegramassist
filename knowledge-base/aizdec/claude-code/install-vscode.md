# Программа VS Code

**Курс:** Claude Code  
**URL:** https://www.aizdec.me/courses/claude-code/install-vscode

---

VS Code это программа, в которой живёт Claude Code. Думай о ней как о рабочем столе для своих проектов. Сама по себе она бесплатная и ставится за 2 минуты.

## **[ ДЛЯ MAC ]**

### **ШАГ 1: Скачать**

1. Открой браузер и перейди на **code.visualstudio.com**

1. Нажми большую синюю кнопку **Download for Mac**

1. Скачается файл типа `VSCode-darwin-arm64.zip` (или похожий)

### **ШАГ 2: Установить**

1. Открой папку **Загрузки** (Downloads)

1. Найди скачанный `.zip` файл и дважды кликни — он распакуется

1. Появится файл **Visual Studio Code** с синей иконкой

1. Перетащи его в папку **Программы** (Applications)

💡 **Зачем в Программы:
**Можно запускать и из Загрузок, но тогда при каждом обновлении будут проблемы. В Программах всё работает как надо.

### **ШАГ 3: Запустить**

1. Открой **Finder** → **Программы** (Applications)

1. Найди **Visual Studio Code** и дважды кликни

1. Может появиться предупреждение "Программа загружена из интернета" — нажми **Открыть.**

**Проверка:**

✅ Открылось окно VS Code с тёмным интерфейсом

✅ Слева панель с иконками

✅ В центре приветственная вкладка

**Всё так? → Переходи к следующему уроку.**

## **[ ДЛЯ WINDOWS ]**

### **ШАГ 1: Скачать**

1. Открой браузер и перейди на **code.visualstudio.com**

1. Нажми большую синюю кнопку **Download for Windows**

1. Скачается файл типа `VSCodeUserSetup-x64.exe`

### **ШАГ 2: Установить**

1. Открой папку **Загрузки**

1. Запусти скачанный `.exe` файл

1. Появится установщик — жми **Далее** на каждом шаге

1. На шаге "Выберите дополнительные задачи" поставь галочку **Добавить в PATH**

1. Нажми **Установить** и дождись завершения

⚠️ **Важно:
**Галочка "Добавить в PATH" — обязательна. Без неё Claude Code не сможет нормально работать. Если пропустил, переустанови.

### **ШАГ 3: Запустить**

1. Нажми **Win** (кнопка с флажком Windows)

1. Нап��ши **VS Code** или **Visual Studio Code**

1. Кликни на появившуюся программу

**Проверка:**

✅ Открылось окно VS Code с тёмным интерфейсом

✅ Слева панель с иконками

✅ В центре приветственная вкладка

**Всё так? → Переходи к следующему уроку.**

💭 **От Тома:
**VS Code это не просто текстовый редактор. Это среда, где ты будешь проводить большую часть времени, работая над проектами. Привыкай к ней. Через пару дней она станет родной.
